fSnippet은 강력한 텍스트 확장 기능을 제공하기 위해 다양한 **플레이스홀더(Placeholder)**를 지원합니다. 스니펫 내에 `{{...}}` 형식으로 작성하면, 텍스트 확장 시점에 동적인 값으로 자동 변환됩니다.

## 📅 1. 날짜 및 시간 (Date & Time)

현재 날짜와 시간을 다양한 형식으로 삽입할 수 있습니다.

| 플레이스홀더               | 설명                   | 출력 예시        |
| :------------------- | :------------------- | :----------- |
| `{{date}}`           | 기본 날짜 (`yyyy-MM-dd`) | `2024-05-20` |
| `{{time}}`           | 기본 시간 (`HH:mm:ss`)   | `14:30:45`   |
| `{{date:short}}`     | 짧은 날짜 (`MM/dd/yy`)   | `05/20/24`   |
| `{{isodate:format}}` | **사용자 지정 포맷**        | (아래 참조)      |
### 사용자 지정 포맷 (Custom Format)
`{{isodate:포맷}}` 형식을 사용하여 원하는 날짜/시간 형식을 자유롭게 지정할 수 있습니다. 포맷은 표준 Unicode Date Format 패턴을 따릅니다.

* **예시**:
    * `{{isodate:yyyy.MM.dd}}` → `2024.05.20`
    * `{{isodate:yyyy년 MM월 dd일 (E)}}` → `2024년 05월 20일 (월)`
    * `{{isodate:HH시 mm분}}` → `14시 30분`
    * `{{isodate:yyyy-MM-dd HH:mm:ss}}` → `2024-05-20 14:30:45`

## 📋 2. 클립보드 (Clipboard)

현재 클립보드에 복사된 내용을 스니펫에 포함할 수 있습니다.

| 플레이스홀더                    | 설명                                                         |
| :------------------------ | :--------------------------------------------------------- |
| `{{clipboard}}`           | 클립보드의 내용을 그대로 삽입합니다.                                       |
| `{{clipboard:trim}}`      | 클립보드 내용의 앞뒤 공백을 제거하고 삽입합니다.                                |
| `{{clipboard:uppercase}}` | 클립보드 내용을 모두 **대문자**로 변환합니다.                                |
| `{{clipboard:lowercase}}` | 클립보드 내용을 모두 **소문자**로 변환합니다.                                |
| `{{clipboard:capitals}}`  | 클립보드 내용의 각 단어 첫 글자를 대문자로 변환합니다 (Capitalize).               |
| `{{clipboard:N}}`         | 클립보드 히스토리의 **N번째** (1~9) 항목을 삽입합니다. (예: `{{clipboard:1}}`) |

## 📍 3. 커서 위치 (Cursor Positioning)

스니펫 확장이 완료된 후, 커서가 위치할 곳을 지정할 수 있습니다.

| 플레이스홀더       | 설명                              |
| :----------- | :------------------------------ |
| `{{cursor}}` | 텍스트 확장이 끝난 후, 커서를 이 위치로 이동시킵니다. |
* **주의**: 스니펫 내에 `{{cursor}}`는 **하나만** 사용해야 합니다. 여러 개가 있을 경우 오류가 발생하거나 의도하지 않게 동작할 수 있습니다.

## ⌨️ 4. 동적 입력 (Dynamic Input)

스니펫 실행 시 사용자로부터 입력을 받아 채워 넣을 수 있는 필드입니다. 스니펫 실행 시 입력 창이 나타납니다.

* **기본 형식**: `{{필드명}}`
* **기본값 포함**: `{{필드명:기본값}}`

**예시**:
```text
안녕하세요, {{name}}님.
오늘 {{meeting:오후 2시}} 미팅 참석 부탁드립니다.
```
위 스니펫을 실행하면 `name`을 입력받는 창과, `meeting`을 입력받는 창(기본값 "오후 2시")이 나타납니다.

## 🔗 5. 고급 참조 (Reference)

다른 스니펫이나 파일의 내용을 현재 스니펫에 포함(Include)할 수 있습니다.

### 5.1 스니펫 참조 (Snippet Reference)
다른 폴더에 있는 스니펫을 가져와서 재사용할 수 있습니다.

* **형식**: `{{폴더명/스니펫이름}}`
* **예시**: `{{Common/Header}}` (Common 폴더의 Header 스니펫 내용을 삽입)

### 5.2 파일 참조 (File Reference)
외부 파일의 텍스트 내용을 불러올 수 있습니다.

| 플레이스홀더                | 설명                 | 예시                         |
| :-------------------- | :----------------- | :------------------------- |
| `{{/path/to/file}}`   | 절대 경로의 파일 내용       | `{{/Users/me/note.txt}}`   |
| `{{~/path/to/file}}`  | 홈 디렉토리(`~`) 기준 경로  | `{{~/Documents/sign.txt}}` |
| `{{./path/to/file}}`  | 현재 스니펫 파일 기준 상대 경로 | `{{./footer.md}}`          |
| `{{../path/to/file}}` | 상위 디렉토리 기준 상대 경로   | `{{../config/style.css}}`  |

## ⚡️ 6. 자동화 동작 (Automated Actions)

스니펫 확장 시 macOS의 **단축어(Shortcuts) 앱**을 실행시킬 수 있습니다.

| 플레이스홀더                    | 설명                                      | 예시                          |
| :------------------------ | :-------------------------------------- | :-------------------------- |
| `{(Shortcut Name)}`       | 지정된 이름의 단축어를 실행하고, 그 결과(텍스트 출력)를 삽입합니다. | `{(My Daily Brief)}`        |
| `{(Shortcut Name:Input)}` | **입력값**을 단축어에 전달하여 실행합니다.               | `{(Translate to EN:안녕하세요)}` |
* **참고**: 단축어 실행 결과가 텍스트인 경우 해당 위치에 삽입됩니다. 결과가 없으면 실행만 됩니다.
